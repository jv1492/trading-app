# Trading App

A starter repository for a trading application.

## Features

- React + TypeScript + Vite frontend
- Sample trading dashboard layout
- Starter files for further development

## Setup

```bash
cd trading-app
npm install
npm run dev
```

## GitHub

To publish this repository to your GitHub account, you can use GitHub CLI if configured:

```bash
gh repo create YOUR_USERNAME/trading-app --public --source=. --remote=origin --push
```

If you prefer, create a repository on GitHub manually and then add it as a remote:

```bash
git remote add origin https://github.com/YOUR_USERNAME/trading-app.git
git push -u origin main
```
